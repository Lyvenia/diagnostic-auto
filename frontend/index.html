🆘 Aide & Support

Retrouvez les réponses aux questions fréquentes ou contactez l'équipe Lyvenia.

Branchez l'adaptateur ELM327 sur le port OBD2 du véhicule (sous le tableau de bord côté conducteur). Allumez le contact sans démarrer le moteur. Dans l'onglet Paramètres, vérifiez que le port COM est correct, puis lancez un diagnostic depuis l'onglet principal.
L'analyse IA nécessite une clé API Anthropic valide. Rendez-vous sur console.anthropic.com pour en obtenir une, puis saisissez-la dans la section "Clé API Anthropic" ci-dessus. Vérifiez également votre connexion internet.
Certains véhicules anciens ou aftermarket ne transmettent pas de VIN. Vous pouvez saisir manuellement la marque, le modèle et l'année via le bouton Véhicule lors du diagnostic. Ces informations enrichissent l'analyse IA.
Après un diagnostic complet avec analyse IA, un bouton 📄 Exporter PDF apparaît dans la barre d'actions. Le rapport inclut les codes DTC, l'analyse, les données temps réel et les recommandations.
Contactez l'équipe Lyvenia pour recevoir la dernière version. Les mises à jour du logiciel sont fournies par votre interlocuteur commercial ou via le support technique.

Lyvenia décline toute responsabilité quant aux décisions techniques, mécaniques ou de sécurité prises sur la seule base des résultats fournis par RODIA. En cas de doute, consultez toujours un garagiste agréé.

Effacer les codes DTC ?

Cette action supprime les codes de défaut mémorisés dans l'ECU. Le bus CAN sera mis en pause le temps de l'opération.

Contact allumé (position ON), moteur tournant ou coupé
Ne pas bouger le câble OBD2 pendant l'opération
L'opération prend environ 5 secondes
Après effacement : roulez 10–15 min pour que l'ECU se recalibre

À noter : certains codes permanents (PDTC) ne peuvent pas être effacés par outil — le calculateur les supprime seul après plusieurs cycles sans défaut.
